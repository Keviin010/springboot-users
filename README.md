# springboot-users

Aplicación web para gestionar usuarios. Permite crear, editar, listar y eliminar usuarios desde el navegador, guardando los datos en una base de datos MySQL.

## Tecnologías

- Java 21
- Spring Boot 3.4.3
- Spring MVC
- Thymeleaf
- Spring Data JPA
- MySQL


## Ejecución
```bash
mvn spring-boot:run
```

Acceder en `http://localhost:8080/app/list`
